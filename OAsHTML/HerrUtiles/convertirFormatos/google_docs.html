Google Docs

Google Docs ofrece un convertido de voz a texto, es un asistente personal que permite dictar un texto usando la voz. 

Para usar esta herramienta es necesario tener una cuenta de Google. Una vez registrado solo es necesario hablar alto y claro y un relativo silencio alrededor. Esta herramienta reconoce hasta 40 idiomas diferentes, lo que resulta muy útil en el aprendizaje de idiomas [1].
