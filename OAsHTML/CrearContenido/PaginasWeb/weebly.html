Weebly

Weebly es una plataforma en línea para crear páginas web, es flexible y fácil de usar. Las páginas web se construyen arrastrados y reordenados los elementos (texto, imágenes, mapas) en columnas y filas en el cuerpo de una página, cada elemento viene con varias opciones de personalización. También hay secciones prediseñadas que puede añadir a las páginas, las secciones son para contenido que requiere varios elementos.

Weebly es el único creador de sitios web que ofrece una tienda de aplicaciones llamada App Center, las cuales añaden nuevas funcionalidades al sitio web.

Características de Weebly:

  • Permite crear blogs.
  • Permite crear tiendas online.
  • Permite crear formularios.
  • Tiene aplicaciones para iPhone, Android, iPad y Apple Watch.
  • Está listo para pantallas retina
  • Tiene un sistema de membresía
  • No permite un diseño desde cero
  • Multi-idioma (complicada configuración)
  • Permite crear menú de restaurante
  • Tiene un reproductor de audio básico con dos skins.
  • Tiene una amplia seleccion de temas [5].
