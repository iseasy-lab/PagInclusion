WordPress

WordPress es una aplicación de software libre para crear y gestionar sitios web, permiten crear webs y sus contenidos de una forma visual, sin tener que programar, lo que permite que prácticamente cualquier persona pueda manejarlo.

WordPress permite crear prácticamente cualquier tipo de web: sitios personales, blogs, webs corporativas, revistas, tiendas online, etc. [3].
