Introducción

El uso de las herramientas de videoconferencia  ofrecen una serie de ventajas y beneficios:
  • Permite que fluya la comunicación, son un buen medio de relación entre docente-estudiante. También es una forma que pueden tener los profesores de conocer mejor a sus alumnos, sus inquietudes, necesidades o problemas.
  • Permite establecer relaciones personales entre los alumnos, permite intercambiar información, gustos y experiencias, permite crear grupos y de esta manera hacen que los estudiantes tengan el sentimiento de pertenencia a una comunidad.
  • Permitir desarrollar nuevas competencias tecnológicas imprescindibles para otros contextos.
  • Permite a los profesores adquirir nuevos conocimientos y habilidades.

El presente Objeto de Aprendizaje propone herramientas de comunicación virtual.
