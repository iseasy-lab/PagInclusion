Jitsi

Jitsi es una aplicación de videoconferencia, VoIP, y mensajería instantánea con aplicaciones nativas para iOS y Android, y con soporte para Windows, Linux y Mac OS X a través de la web.​

Se trata de una herramienta completamente gratuita que permite realizar videollamadas cifradas en tiempo real de forma sencilla y, sobre todo, segura. La objetivo principal de esta herramienta es garantizar la seguridad de la información, puesto que toda la comunicación está cifrada. Para acceder a Jitsi puedes hacerlo a través de Chrome y sin instalar nada adicional en tu ordenador. Además, la aplicación ofrece algunas opciones como la de compartir pantalla y calendario, un chat de texto o la edición de documentos o presentaciones.[1]
